# Kosh Website

## Stack

- **Framework:** Next.js (App Router)
- **Docs:** Fumadocs
- **Styling:** Tailwind CSS
- **Language:** TypeScript

## Design

See `guide.md` for colours, typography, and brand rules.

- **Logo:** `icon.svg` — use with `#C8A45A` (accent) fill on dark backgrounds (`#0F1F1A`), or `#0F1F1A` fill on light
- **Background:** `#0F1F1A`
- **Surface:** `#1A2F28`
- **Accent:** `#C8A45A`
- **Text:** `#F2EDE3`
- **Font:** Geist (primary), Inter (secondary)
